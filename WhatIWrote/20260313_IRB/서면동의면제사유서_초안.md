# 서면동의면제사유서 초안

## 1. 연구과제명

신기후체제하 한국 기후변화 담론 지형의 변동 분석: 유튜브 영상·댓글·행위자 네트워크를 중심으로

## 2. 연구의 개요

본 연구는 유튜브 공식 API와 공개 검색을 통해 기본 검색어 `기후변화`로 검색된 공개 영상 전체와 그 영상들에 달린 공개 댓글, 공개 대댓글, 공개 채널 메타데이터를 수집·정리하여 한국 기후변화 담론의 장기 변동을 분석하는 학위논문 연구이다. 본 연구용 실제 자료 수집은 `IRB 승인 후` 가능한 시점부터 시작하여 별도로 정한 `최종 수집일`까지 일일 API 할당량 범위 안에서 반복 수행하고, 이후 추가 refresh 또는 update 없이 고정된 코퍼스를 분석한다. 연구자는 연구대상자와 직접 접촉하지 않으며, 별도의 설문조사, 면담, FGI, 실험, 참여관찰을 수행하지 않는다.

## 3. 서면동의 면제를 요청하는 사유

### 1. 연구대상자와의 직접적 상호작용이 없음

본 연구는 연구자가 연구대상자에게 직접 접근하거나 접촉하지 않는 공개 온라인 자료 분석 연구이다. 연구대상 자료는 연구 개시 이전에 이미 유튜브 공개 영역에 게시되어 있는 영상 및 댓글이며, 연구자는 게시자에게 참여 요청, 연락, 인터뷰 요청, 추가 정보 요청을 하지 않는다. 따라서 일반적인 서면동의 절차를 실제로 수행할 구조가 존재하지 않는다.

### 2. 연구대상 규모가 매우 크고 동적이어서 개별 동의 획득이 사실상 불가능함

본 연구의 분석대상 규모는 자연인 수가 아니라 공개 YouTube account/channel ID의 유니크 수 및 중복 제거 없는 자료 행 수를 기준으로 설명한다. 내부 추정 메모상 유니크 ID 기준 안전상 상한은 `1,500,000 public YouTube account/channel IDs`이며, 중복 제거 없는 comments 행 기준 안전한 상단 planning 값은 약 `1,800,000 comments`이고, videos 행 기준 상단 planning 값은 약 `42,000 videos`이다. 이처럼 대상군 규모가 매우 크고, 게시물 삭제·비공개 전환·계정 상태 변화가 지속적으로 발생하는 온라인 공개자료 연구에서 개별 게시자 모두에게 서면동의를 받는 것은 현실적으로 불가능하다.

### 3. 개별 동의를 시도하는 행위가 오히려 사생활 침해 및 추가 위험을 유발할 수 있음

본 연구는 공개 게시물을 보수적으로 처리하기 위해 비식별화, 가명처리, 직접 인용 최소화, 원자료 재배포 금지 등의 조치를 취하고자 한다. 반대로 개별 동의를 받기 위해 게시자에게 연락을 시도할 경우, 연구자가 개별 계정에 직접 접근하게 되고, 이는 원래 연구자가 접촉하지 않아도 될 게시자에게 불필요한 접촉과 사생활 침해 가능성을 발생시킬 수 있다. 따라서 동의 획득 시도 자체가 연구대상자 권리 보호에 반드시 더 유리하다고 보기 어렵다.

### 4. 서면동의 면제가 연구대상자의 권리와 복지를 해하지 않도록 보호조치를 설계함

본 연구는 다음과 같은 방식으로 연구대상자의 권리와 복지를 보호한다.

- 공개 영역 자료만 수집한다.
- 유튜브 공식 API 및 관련 개발자 정책 범위 안에서만 자료를 수집하며, 비공식 스크래핑이나 우회 수집을 하지 않는다.
- 시청각 원본 파일은 다운로드·저장하지 않고, 공개 메타데이터와 공개 텍스트만을 취급한다.
- 분석용 데이터에서는 account/channel ID를 연구용 가명 코드로 대체한다.
- 사용자명, 채널명, URL, 프로필 이미지, 계정 식별자 등 직접 식별 가능 정보는 결과물에 제시하지 않는다.
- 댓글 원문 직접 인용은 원칙적으로 하지 않으며, 필요시에도 비검색 가능 형태의 의역 또는 요약을 우선 사용한다.
- 스크린샷은 원칙적으로 사용하지 않는다.
- 댓글 텍스트에 포함된 전화번호, 이메일 주소, 실명 등 직접 식별 단서는 분석 전 제거 또는 마스킹한다.
- 원자료와 분석자료를 분리하고, 재식별 가능 키 파일은 연구책임자 `박정현`과 공동연구자 `윤순진`만 접근 가능한 상태로 별도 관리한다.
- 승인되지 않은 API 원자료, 식별 가능한 원자료, 원문 댓글은 `최종 수집일`로부터 `30일` 이내 삭제한다.
- 장기 보관 대상은 직접 식별자와 원문 댓글을 제거하고, 개별 댓글 복원이 어려운 수준으로 가공된 분석산출물, 코드북, 집계표, 통계·네트워크 요약 지표, 분석 스크립트, 재현성 문서로 한정한다.
- 재식별 가능 키 파일은 논문 심사 종료 및 최종 분석 검증 완료 후 즉시 파기한다.

위와 같은 조치를 고려할 때, 서면동의 면제는 연구대상자의 권리와 복지를 침해하지 않는 범위 내에서 필요한 예외로 볼 수 있다.

## 4. 연구자가 접촉하지 않는 이유

본 연구는 공개 온라인 담론의 자연발생적 구조를 분석하는 연구이므로, 연구자가 개별 게시자에게 접촉하지 않는 것이 연구설계상으로도 적절하다. 연구자가 게시자에게 별도로 연락하는 경우, 본래 연구대상에 존재하지 않던 상호작용이 생기게 되어 연구윤리적 부담뿐 아니라 자료의 자연발생성을 해칠 우려도 있다.

## 5. 결론

본 연구는 공개 온라인 자료를 이용하는 비개입 연구로서 연구대상자와의 직접 접촉이 없고, 대상군 규모가 유니크 ID 기준으로도 매우 크며 중복 제거 없는 comments 행 기준으로는 더욱 큰 규모가 될 수 있어, 개별 서면동의를 현실적으로 받기 어렵다. 또한 개별 동의를 시도하는 행위가 오히려 추가적인 사생활 침해 또는 불필요한 접촉 위험을 발생시킬 수 있다. 이에 본 연구는 일반심의로 제출하되, 연구대상자의 권리와 복지를 보호하기 위한 비식별화·가명처리·인용 최소화·접근통제·파기 계획을 전제로 서면동의 면제를 요청하고자 한다.
