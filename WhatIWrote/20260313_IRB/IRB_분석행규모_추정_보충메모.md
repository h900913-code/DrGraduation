# IRB 분석행 규모 추정 보충 메모

## 1. 목적

이 메모는 IRB 문서에서 `연구대상 수`를 유니크 작성자 ID 기준으로 볼지, 아니면 중복 제거 없는 실제 자료 행 수 기준으로 볼지 불명확할 수 있다는 점을 보완하기 위해 작성한 내부 작업 메모이다.

기존 `WhatIWrote\20260313 irb_target_population_estimate.md`가 `public YouTube account/channel IDs`의 유니크 수를 기준으로 한 추정이라면, 이 메모는 `videos` 테이블 행 수와 `comments` 테이블 행 수 자체를 기준으로 한 추가 추정이다.

이 메모의 숫자는 원래 `2026-01-27` 종료점을 중심으로 한 내부 추정에 가깝다. 이후 IRB 문서에서는 자료 수집 및 분석 범위를 앞뒤 `2개월` 완충을 둔 `2015-10-12 ~ 2026-03-27` 전체로 확장하였다. 따라서 아래 수치는 확장된 완충 구간 전체를 재산정한 값이 아니라, 현재 시점에서 사용할 수 있는 보수적 planning estimate로 읽는 편이 안전하다.

## 2. 기준 시점

- 산정 기준 시점: `2026-03-13`
- 기준 DB 상태:
  - 현재 `videos` 행 수: `12,237`
  - 현재 `comments` 행 수: `537,695`

여기서 `comments`는 댓글과 대댓글 행 수를 모두 포함한다. 동일인이 여러 번 작성한 댓글도 전부 포함된 값이다.

## 3. 미검색 구간과 앵커 월

- 큰 미검색 구간: `2022-02 ~ 2026-01`의 `48개월`
- 앵커 월:
  - `2022-01`: `199 videos`, `6,365 comments`
  - `2026-02`: `1,023 videos`, `45,360 comments`

`2022-01`은 partial month이므로 하한에 가까운 값으로 해석한다.

## 4. 시나리오별 추정

- 보수적 plateau(2021 수준 유지): `24,477 videos`, `824,585 comments`
- 완만한 성장(geometric): `36,490 videos`, `1,488,393 comments`
- 절충 성장(hybrid): `39,128 videos`, `1,635,469 comments`
- 강한 선형 성장(linear): `41,765 videos`, `1,782,545 comments`

## 5. 해석과 권고값

한 줄 요약은 다음과 같다.

- 보수적으로는 약 `2.4만 영상 / 82만 댓글`
- 성장 지속 가정이면 약 `3.9만 영상 / 160만 댓글`
- 안전한 상단 planning 값으로는 약 `4.2만 영상 / 180만 댓글`

IRB 초안에 반영할 때는 다음과 같이 쓰는 것이 적절하다.

- videos 행 기준 중심 추정 참고치: 약 `39,128`
- videos 행 기준 안전한 상단 planning 값: 약 `42,000`
- comments 행 기준 중심 추정 참고치: 약 `1,635,469`
- comments 행 기준 안전한 상단 planning 값: 약 `1,800,000`
- 행정 입력용 보수적 단일 수치 권고값: `2,000,000`

## 6. IRB 문서 적용 원칙

- `유니크 ID 기준`과 `행 기준`을 구분해서 병기한다.
- `유니크 ID 기준`은 `1,500,000 public YouTube account/channel IDs`를 안전상 상한으로 유지한다.
- `중복 제거 없는 comments 행 기준`은 `1,800,000 comments`를 안전한 상단 planning 값으로 병기한다.
- IRB 시스템이 단일 수치만 입력받고 심의자가 자료 행 수 기준으로 읽을 가능성까지 고려해야 하는 경우에는, 입력 편의상 더 보수적인 단일 입력 권고값으로 `2,000,000`을 사용할 수 있다.
- 단, 영상 행 수는 comments 행 수에 비해 규모가 작으므로 단일 입력란에서는 comments 기준 상한을 우선 사용하고, videos 행 수는 본문 설명에 별도로 병기하는 방식이 실무상 더 자연스럽다.

## 7. 범위 확장에 따른 해석 주의

- 현재 행 규모 추정은 `2015-12-12 ~ 2026-01-27` 중심 구간에 맞춰 만들어진 planning estimate에 가깝다.
- IRB 문서상 자료 수집 범위를 `2015-10-12 ~ 2026-03-27`로 확장한 효과는 아직 별도 재산정하지 않았다.
- 따라서 이 메모의 수치는 `전후 2개월 완충 구간을 포함한 확정 계수`가 아니라, 현재 단계의 보수적 설명용 수치로 사용한다.
- 추후 완충 구간까지 포함한 정확한 행 수가 필요해지면 별도 재산정이 필요하다.
