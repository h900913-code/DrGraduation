# 학위종별 변경 즉시 실행 체크리스트

마지막 업데이트: 2026-03-09

출처 코드 안내: [source_index.md](/C:/Repositories/20260309_DrGraduationAdminAssist/GraduationAssistant/source_index.md)

전제:

- 학적변동은 아직 안 되었음
- 따라서 현재 핵심 절차는 `학위종별 변경 신청`
근거: 사용자 진술, `Q1, G1, G4`

## 바로 해야 할 일

1. `박사논문 요지발표 신청서`를 준비한다.
2. `학위 종별 변경 사유서` 초안을 검토한다.
3. `분야 인정 가능 교과목 확인 명세서` 초안을 검토한다.
4. 지도교수와 `요지발표 + 1차심사 통합 진행 여부`를 확인한다.
5. 심사위원 구성에서 `환경관리학과 교수 1인 이상` 포함 가능 여부를 확인한다.

## 서류별 제출 시점

### 1. 요지발표 신청서

- 창구: `학과사무실`
- 시점: `3월 중`
- 메모: QnA상 `4월 초 학과사무실이 수합 후 행정실 제출 예정`
근거: `Q1, F3, G3`

### 2. 학위종별 변경 서류

- 서류:
  - `학위 종별 변경 사유서`
  - `분야 인정 가능 교과목 확인 명세서`
- 창구: `학과사무실`
- 시점: `논문심사 신청기간`
- 메모:
  - QnA 기준 `1학기 4월 중순`
  - 2학기는 기존 규정 흐름상 `10월 중순` 가능성이 높지만, 실제 공지로 재확인 필요
근거: `Q1, G1, G3, F1, F2, W2`
주의: `2학기 10월 중순`은 `Q1`의 1학기 답변과 기존 학기 운영 흐름을 바탕으로 한 추정입니다.

### 3. 심사 관련 서류

- 서류:
  - `논문심사위원추천서`
  - 심사료 관련 서류
  - 기타 심사요구서류
- 창구: `행정실`
- 시점: `논문심사 신청기간`
근거: `Q1, G3, F4, F5, F6`
