## 1. 문의1: 박사논문 요지발표, 1차 심사, 변경서류 제출 시점

### 사용자가 물은 내용

- 박사논문 `요지발표`와 `1차 심사`를 반드시 분리해야 하는지
- `박사논문 요지발표 신청서`를 언제, 어디에 내는지
- `논문심사위원추천서`, `학위명칭 변경 사유서`, `환경관리학 분야 인정 가능 교과목 확인 명세서`를 `요지발표 신청 시점`에 함께 내야 하는지

### 학과사무실 답변 핵심

- `내규상 요지발표와 1차심사가 구분되어있지만 보통 둘을 합쳐서 진행`
- `이건 지도교수님과 논의해서 진행`
- `박사논문요지발표 신청서는 작성해서 3월 중으로 제게 제출`
- `4월 초에 요지발표신청서를 수합하여 행정실에 제출 예정`
- `학위종별 변경 서류`
  - `논문심사 신청기간(1학기 4월 중순)에 학과사무실로 제출`
- `논문 심사원 제출 및 심사료 납부 관련 서류(논문심사위원추천서 포함)`
  - `논문심사 신청기간(1학기 4월 중순)에 행정실로 제출`
근거: `Q1`

### 의미

- `요지발표 신청서`는 `3월 중`
- `학위종별 변경 사유서`, `분야 인정 가능 교과목 확인 명세서`는 `4월 중순 논문심사 신청기간`
- `논문심사위원추천서`도 `4월 중순`
- 따라서 `요지발표 신청 시점에 모든 서류를 다 낼 필요는 없다`

## 2. 문의2: 사회과학/이공계열 요건이 21학번에도 적용되는지

### 사용자가 물은 내용

- 환경관리학 박사 학위 수여 요건에 적힌 `사회과학 계열 과목 1과목 + 이공계열 과목 1과목` 또는 계열별 최소학점 조항이 `2021학년도 2학기 입학생`에게도 적용되는지

### 학과사무실 답변 원문 핵심

- `계열별 최소 3학점 이수 조항은 24학번부터 적용되는 것으로, 21학번인 선생님은 해당되지 않습니다.`
근거: `Q2`

### 의미

- 당신은 `2021학년도 2학기 입학`이므로 이 `계열별 조항` 때문에 막히지 않습니다.
- 기존에 남아 있던 `사회과학/이공계열 요건` 불확실성은 사실상 해소됐습니다.

## 3. 이 QnA가 기존 판단을 어떻게 바꾸는지

### 강화된 결론

- `환경관리학 박사 학위변경의 가장 큰 리스크 중 하나였던 계열 요건 불확실성`이 크게 줄었습니다.
- 이제 남은 핵심은 아래입니다.
  - `학적변경`이 이미 된 것인지
  - 아니면 `학위종별 변경`만 하면 되는지
  - `분야 인정 가능 교과목 확인 명세서`에 어떤 과목을 넣을지
  - `환경관리학과 교수 1인 이상` 심사위원 구성을 어떻게 맞출지
근거: `Q1, Q2, G1, G3`

### 바뀐 실무 우선순위

1. 학적변동 상태 확인
2. 분야 인정 가능 교과목 확인 명세서 초안 작성
3. 학위명칭 변경 사유서 초안 작성
4. 지도교수와 심사위원 구성 논의
5. 논문 요지발표 일정 맞추기

## 4. 현재 한줄 결론

`QnA` 기준으로 보면, 당신의 졸업 리스크는 이제 `계열 요건` 문제가 아니라 `행정 서류 정리 + 심사 일정 대응 + 심사위원 구성` 쪽으로 좁혀졌습니다.
근거: `Q1, Q2, G3`
